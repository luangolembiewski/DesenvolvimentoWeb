Bootstrap

Oque é:

Framework que disponibiliza várias classes pré-definidas de css, que fazem com que o site seja responsivo a qualquer tipo de tela.
CSS: fim de estilizar os elementos de nossas páginas de forma padronizada, responsiva e ágil.
JavaScript: teremos a possibilidade de tornar os elementos dinâmicos ou com animação

    Oque é:

    Uma classe CSS, responsável por definir um layout estrutural, proporcionalmente alinhado na página

    Ao utilizar a classe "container" o bloco automaticamente já recebe uma estilização baseada na resolução da tela.

    Container-fluid

    Define um layout que considera a largura total da página

    Por padrão, utilizamos sempre um elemento pai com a classe CONTAINER, seja uma div ou uma section, onde iremos colocar os demais elementos.

    É usado também as classes: row e col, para definir que os elementos irão ter os comportamentos de linha e de coluna

    Classes col prontas no bootstrap

  • Border: define as bordas do elemento.
  • Border-danger: define as bordas com a cor vermelha.

    • Jumbotron

    Componente usado para chamar mais atenção

    Componentes

  • Jumbotron: define que o elemento será um jumbotron e, por esse motivo, aplica um estilo diferente do container, possibilitando a cor cinza no fundo e toda a largura da tela.
  • Display-4: a classe "display" define cabeçalhos maiores e levemente mais estilizados, onde você escolhe a densidade entre 1 a 4. Funciona de forma semelhante à hierarquia de título do HTML, onde temos de h1 até h6.
  • Lead: define um tipo de parágrafo mais destacado.
  • my-4: nessa classe temos a letra "m" para margem e a letra "y" para indicar que a margem deve ser aplicada tanto na parte superior quanto na inferior, e, por fim, o número que "4" que indica a medida (o número 4 refere-se a um padrão já definido pelo Bootstrap).
  • btn, btn-primary e btn-lg: definem um botão, a cor de fundo do botão e o tamanho do botão, respectivamente.

    Tabelas

    Criar tabela responsiva e padronizada

    Se existirem muitas colunas ou o conteúdo da coluna for extenso e você queira exibir sem que haja o encolhimento desse conteúdo, utilize a classe "table-responsive" envolvendo o elemento com a classe "table":

    <div class="table-responsive">
    <table class="table">
    ...
    </table>
    </div>

    Formulario

    Veja que, ao usar o Bootstrap em formulários, você irá se preocupar mais com a função de cada componente do que com o visual e com o estilo CSS. Como vimos, basta envolver os componentes que devam ficar numa linha do formulário com uma div e a classe "form-group" e colocar a classe "form-control" em cada elemento input.

    Logo, temos as seguintes classes:
  • form-group: Usada na
    , agrupa os elementos label e input em linha.
  • form-control: Usada em controles (inputs), define seus estilos e alinhamentos, juntamente com seu label associado.
  • form-check: Usada quando o elemento é do tipo "check", deixando os elementos na mesma linha.

    • Botões

    Serve para estilizar botões para ficar mais elegantes e alinhados

    a classe "btn"serve tanto para <buttons> quanto para <a> e <input>

    Tamanho do botão

  • btg-lg: define o botão com tamanho grande(large).
  • btg-sm: define o botão com tamanho pequeno(small).
  • btn-block: define o botão para ocupar toda a largura do elemento pai.

    • Navbar

    Componentes:

  • navbar: Define o elemento como uma navbar.
  • navbar{-dark-light}: Define a cor dos itens textuais inclusos na navbar. Logo, se sua navbar tiver o plano de fundo escuro, o ideal é utilizar o "navbar-dark", pois a cor do texto e nome dos itens, por exemplo, serão brancos.
  • bg-primary: Define a cor do plano de fundo da navbar.
