    Overflow

    Visible:

    Essa Declaração defini que mesmo que o conteúdo não caiba ele aparecerá mesmo assim.

    Hidden:

    Essa Declaração defini que o conteúdo que couber será apresentado, já o conteúdo que exceder será oculto.

    Scroll:

    Essa Declaração sempre exibe uma barra de scroll para que o usuario visualize mesmo que não exceda

    Auto:

    Essa Declaração caso o conteúdo exceda, exibe uma barra de scroll para que o usuario visualize
