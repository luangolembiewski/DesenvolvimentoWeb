    Async/defer

    Problema:

  • Problema do site pausar para executar o Script no meio da aplicação

  • Executa tudo de forma assíncrona o download e execução

  • O download do Script é feito junto ao carrgamento da página, Porém a execução é feita ao final

    • KeyWords

  • VAR -> (Escopo global é acessada de qualquer local dentro da aplicação e pode ser chamada até mesmo antes da sua criação)
  • LET -> (Não permite que a variavel seja chamada antes de sua criação, Como acontece com Var), (não permite que a variavel seja acessada fora do seu escopo: escopo local)
  • CONST -> possui a segurança da let, porem não pode ser modificada, sendo associada ao primeiro valor de criação (só é possivel alterar seu valor em nível secundario)

    JSON

    Apresentação: uma redução de Javascript: o JSON é utilizado em todas as linguagens de programação, inclusive em bancos de dados. para trocas de dados simples.

    Com ele, é possível representar e transferir dados entre clientes e servidores, ou ainda, entre aplicações que precisam se comunicar. Os dados ficam estruturados em hierarquias e armazenados em pares de chave-valor.

    Propriedades: Ao atibuir json a uma variável, ela se torna um objeto / Como objeto Json, ela herda todos os objetos da classe mãe / cada atributo do JSON torna-se um atributo da variável

    Partes Json

    {} > As chaves {} delimitam um objeto. Logo, no exemplo anterior, temos um objeto.

    : > Os dois pontos “:” separam chaves e valores.

    , > As vírgulas "," separam os pares de chaves/valores.

    [] > Os colchetes [] delimitam um array
